# Revisador de Medios

> If you speak english, please read the [english version](https://github.com/Exatoon27/mediareview/blob/master/README.en.md).

Una herramienta simple para revisar y clasificar archivos multimedia (imágenes y videos) rápidamente. Permite guardar los archivos revisados en un archivo ZIP y extraer fotogramas de videos.

## Características

- Visualización rápida de imágenes y videos.
- Clasificación de archivos como "guardar" o "descartar".
- Extracción de fotogramas de videos.
- Descarga de archivos clasificados en un archivo ZIP.

## Uso

### Local

1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador web.
3. Arrastra y suelta tus archivos multimedia en la interfaz.
4. Usa los botones para clasificar los archivos:
   - **Guardar**: Marca el archivo para guardar.
   - **Descartar**: Marca el archivo para descartar.
   - **Extraer Fotograma**: Extrae un fotograma del video actual (solo para videos).
5. Una vez que hayas revisado todos los archivos, haz clic en "Descargar Clasificados" para obtener un archivo ZIP con los archivos guardados.

### En línea

También puedes usar la versión en línea de la herramienta en: [MediaReview en línea](https://exatoon27.github.io/MediaReview/)

## Requisitos

- Un navegador web moderno (Chrome, Firefox, Safari, Edge).
- Soporte para HTML5 y JavaScript.
